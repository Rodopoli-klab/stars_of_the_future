# stars_of_the_future
Η ομάδα «Αστεράκια του Μέλλοντος» αξιοποιεί ανοιχτή τεχνολογία και απλό αυτοματισμό, μετατρέποντας τα σχολικά κτήρια σε ενεργά πεδία βιωματικής μάθησης. Το έργο υλοποιείται σε σύμπραξη του Νηπιαγωγείου Ροδόπολης και του 2ου Νηπιαγωγείου Άνοιξης, καλλιεργώντας τη συνεργασία και την κουλτούρα ανοιχτής, συμμετοχικής καινοτομίας.
